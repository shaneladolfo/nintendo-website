Navigation Bars

The navigation bar must be included on all pages using this design system. The navigation bar includes a navigation menu and allows users to navigate to the "About" and "Work" pages of the website.

Navigation Bar Example

This navigation bar example is responsive. Adjust the browser size to view responsive sizing.

  • About

  • Work

Accessibility Considerations

Be sure to use the colors specified in the color palette section to ensure accessible contrast ratio. Icons should include descriptive alt attributes that tell users where the link will direct them to.

Usage

  • A navigation bar must be included in all pages created within this design system.

Code Example
