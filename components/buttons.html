Buttons

This design system includes one button, with three different states.

Primary

The primary button is the button that users will encounter the most on the website.

Code

            
              <button href="#" class="btn-primary">Primary</button>

Hover

The hover state of the button will darken to ensure the users will know the button is about to be selected.

Code

             
In CSS:

             
               <.btn-default:hover{}>

Disabled

The disabled button will be unusable and unclickable. Users will rarely encounter a disabled button on this website.

Code

             
               <button href="#" class="btn-disabled">Disabled</button>

Accessibility Considerations

Ensure all buttons are correctly labelled. Make sure all buttons have a link that leads users to the correct page.

Usage

Use buttons to initiate any action on the website
